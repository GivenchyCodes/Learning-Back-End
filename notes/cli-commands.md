pwd: Shows the path of the current directory you are in
ls: Lists all files and folders in the directory
cd: Changes the directory to a new location
mkdir: Creates a new folder/directory
touch: Creates a new empty file
cp: Copies a file or directory
mv: Moves or renames a file or directory
rm: Removes (deletes) a file
cat: Displays the full content of a file
clear: Clears the terminal screen for a fresh start
